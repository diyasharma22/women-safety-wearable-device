Women Safety Wearable Device 🚨

IoT-based emergency SOS and GPS tracking wearable using ESP32, SIM800L, and Neo-6M GPS.

📌 Project Features

  • Emergency SOS Alert System
  • Real-time GPS Tracking
  • SMS Notification using GSM
  • ESP32-based Wearable Device
  • Portable Hardware Prototype
